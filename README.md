# Contributor flow

1. Make local edits
2. Commit changes with change log

```
git add .
git commit
```

3. Pull repo changes and resolve merge conflicts

```
git pull
```

3. Push changes

```
git push
```

# Directory Structure

| Directory   | Usage                       |
| ----------- | --------------------------- |
| src         | HAS Source Files            |
| distrib     | Build output                |
| \<dir>/auto | Auto-generated output files |
